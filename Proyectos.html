JavaScript Ahorcado

El juego de ahorcado desarrollado en JavaScript me brindó una comprensión profunda de la lógica de programación y el manejo de eventos en el navegador. Uno de los desafíos principales fue implementar una interfaz interactiva y responsiva que permitiera al usuario jugar sin problemas. Utilicé técnicas avanzadas de manipulación del DOM y control de eventos para asegurar una experiencia de usuario fluida. Este proyecto no solo mejoró mis habilidades en JavaScript, sino que también me ayudó a enamorarme de la programación.

Scratch Pacman

El proyecto de Pac-Man en Scratch me permitió explorar la lógica detrás de los juegos y cómo se puede implementar de manera visual y accesible. Los desafíos incluyeron la creación de un sistema de colisiones preciso y la programación de la IA de los fantasmas. Utilicé bloques de código para diseñar los movimientos del jugador y los enemigos, y para gestionar los puntos y el tiempo de juego. Este proyecto me ayudó a fortalecer mi lógica algorítmica y mi capacidad para diseñar mecánicas de juego efectivas.

React GifApp

En este proyecto, desarrollé una aplicación para buscar y mostrar GIFs utilizando la API de Giphy. Uno de los principales desafíos fue integrar la API de manera efectiva y gestionar el estado de la aplicación con React. Implementé componentes funcionales y utilicé hooks para manejar la lógica y el estado de la aplicación. La experiencia de trabajar con React y APIs externas me permitió comprender mejor el ciclo de vida de los componentes y mejorar mis habilidades en el desarrollo de aplicaciones web modernas.
